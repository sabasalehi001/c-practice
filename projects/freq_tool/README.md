# Word Frequency Counter

This simple C program reads a text file and counts the frequency of each word.

## Build

```bash
make
```

## Run

```bash
./app input.txt
```

Where `input.txt` is the text file you want to analyze.  The program will print a list of words and their frequencies to the console.

Example `input.txt`:

```
This is a simple example. This is a test.
```

Example output:

```
this: 2
is: 2
a: 2
simple: 1
example: 1
test: 1
```
