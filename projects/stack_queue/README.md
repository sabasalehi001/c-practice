# Simple Stack and Queue Library Demo

This project demonstrates a basic stack and queue implementation in C.

## Building

To build the project, use the following command:

```bash
make
```

This will compile the source code and create an executable named `app`.

## Running

To run the executable, use the following command:

```bash
./app
```

## Example Output

The program will demonstrate the basic operations of a stack and a queue.  The output will show items being pushed onto the stack and popped off, as well as items being enqueued and dequeued from the queue.

```
Stack:
Pushing: 1
Pushing: 2
Pushing: 3
Popping: 3
Popping: 2
Popping: 1

Queue:
Enqueuing: 10
Enqueuing: 20
Enqueuing: 30
Dequeuing: 10
Dequeuing: 20
Dequeuing: 30
```
