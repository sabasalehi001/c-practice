# Data Structures Demo (Stack and Queue)

This project demonstrates a simple stack and queue library implemented in C.

## Build

To build the project, use the following command:

```bash
make
```

This will compile the `main.c` file and create an executable named `app`.

## Run

To run the compiled executable, use the following command:

```bash
./app
```

## Example

The program will demonstrate basic stack and queue operations, such as pushing/popping elements from the stack and enqueueing/dequeueing elements from the queue.  It will print the results of these operations to the console.

Example output:

```
Stack: Push 10
Stack: Push 20
Stack: Pop: 20
Queue: Enqueue 100
Queue: Enqueue 200
Queue: Dequeue: 100
```
