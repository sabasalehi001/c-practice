# Prime Number Toolkit

A simple C project for prime number related utilities, including Sieve of Eratosthenes and prime factorization.

## Building

To build the project, use the following command:

```bash
make
```

This will compile the `main.c` file and create an executable named `app`.

## Running

To run the program, use the following command:

```bash
./app
```

This will prompt you to choose between the Sieve of Eratosthenes and Prime Factorization.  Follow the prompts to enter the necessary input.

### Example

To find all primes up to 50 using the Sieve of Eratosthenes:

1.  Run `./app`
2.  Enter `1` (for Sieve of Eratosthenes)
3.  Enter `50`

The program will then print all prime numbers up to 50.
