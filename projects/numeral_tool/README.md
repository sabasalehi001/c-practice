# Roman Numeral Converter

This project provides a simple command-line tool to convert integers (1-3999) to Roman numerals.

## Build

To build the project, use the following command:

```bash
make
```

This will compile the `main.c` file and create an executable named `app`.

## Run

To run the program, use the following command:

```bash
./app <integer>
```

where `<integer>` is the integer you want to convert to a Roman numeral. The integer must be between 1 and 3999 inclusive.

## Example

```bash
./app 1984
```

Output:

```
MCMLXXXIV
```
