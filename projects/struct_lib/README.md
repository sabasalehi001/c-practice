# Simple Stack and Queue Library Demo

This project demonstrates a basic stack and queue implementation in C.

## Building

To build the project, use the following command:

```bash
make
```

This will create an executable named `app`.

## Running

To run the program, use the following command:

```bash
./app
```

## Example

The program will demonstrate pushing/popping elements on a stack and enqueuing/dequeuing elements from a queue.

Example output:

```
Stack:
Popped: 3
Popped: 2
Popped: 1
Queue:
Dequeued: 1
Dequeued: 2
Dequeued: 3
```
